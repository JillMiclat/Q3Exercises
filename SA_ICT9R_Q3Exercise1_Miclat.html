HTML

HyperText Markup Language

The preferred markup language for building Web pages is HTML. It is the collection of markup symbols or codes added to a file with the purpose of displaying it online. HTML has taught me the basics of web page coding and more.

CSS

Cascading Style Sheets

Is a stylesheet language that is used to describe the presentation of an HTML document. CSS is a language for describing the presentation of Web pages, including colors, layout, and fonts. CSS saves a lot of time,It is capable of controlling the layout of multiple web pages at the same time.

Bootstrap

Bootstrap Version 4

If CSS is used to design and control multiple web pages in one layout, Bootstrap ensures that your file is accessible and appears well in all screens sizes and types of devices. Bootstrap is a front-end programming framework for building free and open-source websites and web applications. Bootstrap is a collection of template design syntax that is intended to make it possible to construct responsive websites that prioritize mobile users.
